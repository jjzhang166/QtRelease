# 离线地图下载工具（瓦片地图）
QML中加载离线瓦片地图时，就需要将瓦片地图下载到本地，并按照规定的瓦片命名方式进行命名。因此我制作了该瓦片地图下载工具，该工具可选择不同地图类型进行下载。</br>
